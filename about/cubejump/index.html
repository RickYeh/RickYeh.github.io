CubeJump

Addictive and challenging side-scrolling jumping game using the Phaser game engine.

Directions: Simply press up, click the mouse, or tap your mobile screen to jump. You can also double jump to clear longer obstacles. Good luck!

CubeJump is fun side-scrolling game I made utilizing the Phaser game engine. Until this, I had only created turn based games and I wanted to make something faster and more challenging.

Tech Stack: Phaser, jQuery, HTML5, Javascript
